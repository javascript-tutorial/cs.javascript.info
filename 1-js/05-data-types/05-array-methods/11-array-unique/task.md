# Filtrace unikátních prvků pole

Nechť `pole` je nějaké pole.

Vytvořte funkci `unikát(pole)`, která vrátí pole s unikátními prvky `pole`.

Příklad:

```js
function unikát(pole) {
  /* váš kód */
}

let řetězce = ["Haré", "Kršna", "Haré", "Kršna",
  "Kršna", "Kršna", "Haré", "Haré", ":-O"
];
```
