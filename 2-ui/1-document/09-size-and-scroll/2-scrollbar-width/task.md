# Jaká je šířka posuvníku?

Napište kód, který vrátí šířku standardního posuvníku.

Ve Windows se obvykle pohybuje mezi `12px` a `20px`. Pokud si pro něj prohlížeč nerezervuje žádné místo (posuvník je poloprůsvitný nad textem, což se také někdy stává), může být `0px`.

P.S. Kód by měl fungovat pro jakýkoli HTML dokument nezávisle na jeho obsahu.
