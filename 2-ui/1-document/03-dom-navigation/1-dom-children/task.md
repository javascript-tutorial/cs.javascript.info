# Děti DOMu

Podívejte se na tuto stránku:

```html
<html>
<body>
  <div>Uživatelé:</div>
  <ul>
    <li>Jan</li>
    <li>Petr</li>
  </ul>
</body>
</html>
```

Pro každý z následujících uzlů uveďte aspoň jeden způsob, jak k němu přistoupit:
- DOM uzel `<div>`?
- DOM uzel `<ul>`?
- Druhý uzel `<li>` (obsahující Petra)?
