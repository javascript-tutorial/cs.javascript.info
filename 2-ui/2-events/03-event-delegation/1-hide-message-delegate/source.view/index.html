Kůň

Kůň je jedním ze dvou existujících poddruhů Equus ferus. Je to lichokopytník patřící do taxonomické čeledi koňovitých. Kůň se během posledních 45 až 55 milionů let vyvinul z malého vícekopytníka Eohippus do dnešního velkého jednokopytníka.

Osel

Osel (Equus africanus asinus) je domestikovaný člen čeledi koňovitých (Equidae). Divokým předkem osla je africký divoký osel (E. africanus). Osel se jako pracovní zvíře používá již nejméně 5000 let.

Kočka

Kočka domácí (latinsky: Felis catus) je malý, zpravidla chlupatý, masožravý savec. Často se jim říká domácí kočky, pokud jsou chovány jako domácí mazlíčci, nebo prostě kočky, pokud není třeba je odlišovat od jiných kočkovitých šelem. Kočky jsou lidmi často oceňovány pro jejich přítulnost a schopnost lovit škůdce.
