Byla jednou jedna prasečí maminka a ta měla tři malá prasátka.

Jednoho dne už byla prasátka tak velká, že jim maminka řekla: „Už jste moc velcí na to, abyste bydleli pořád tady. Musíte jít a postavit si své vlastní domečky. Ale dávejte pozor, aby vás nechytil vlk.“

Tři prasátka se vydala na cestu. „Dáme si pozor, aby nás vlk nechytil,“ řekla.
